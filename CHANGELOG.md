# Changelog

## 0.2.0

- Local syntax validation (no server needed)
- Recursive descent parser and lexer for SAIL expressions
- Errors show as red squiggly underlines as you type
- Validates on open, save, and change (debounced 500ms)
- 99 tests covering lexer, parser, and edge cases

## 0.1.0

Initial release.

- Syntax highlighting for SAIL expressions
- Document formatting
- Block comment toggling
- Bracket auto-closing and matching
